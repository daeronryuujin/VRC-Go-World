# Moderation Presets

## Preset A: Private Club
- Video control: trusted role only.
- Board reset: anyone in match group.
- Rate limits: relaxed.
- Best for scheduled groups.

## Preset B: Semi-Public
- Video control: owner + designated moderators.
- Board reset: vote (>=60% nearby players).
- Rate limits: medium.
- Recommended default.

## Preset C: Public Open Lobby
- Video control: owner/mod only, URL domain allowlist optional.
- Board reset: owner/mod only with cooldown.
- Rate limits: strict.
- Anti-grief UX warnings enabled.

## Recommended Defaults
- Start with **Semi-Public**.
- Elevate to Public preset once analytics show abuse patterns.
